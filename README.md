# Project Overview

The Weather Forecast Application is designed to help users quickly access weather information in a 
clean and interactive interface.

Users can:

- Search weather by city name
- Get weather using their current location
- View temperature, humidity, wind speed, and weather condition
- Toggle between Celsius and Fahrenheit
- View a 5-day extended forecast
- See dynamic background changes based on weather
- Access recently searched cities

The application focuses on:

- User experience
- Responsive design
- Real-time API integration
- Client-side data storage
